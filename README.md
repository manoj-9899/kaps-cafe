## About This Website

This is the official website for Kaps Café, Mahad. It is a static website meaning it runs directly in a browser with no server or database required.

## How to Open the Website Locally

Open the kaps-cafe folder.
Double click index.html.
It opens in your browser.

The homepage shows a short menu preview. The full menu with search and all categories is on menu.html.

Note: For the Google Maps embed to work correctly you need an internet connection.

## How to Put It Live on the Internet

Recommended hosting: Netlify (free)

1. Go to netlify.com
2. Create a free account
3. Drag and drop the kaps-cafe folder onto the Netlify dashboard
4. Your site is live instantly

To connect your own domain name contact your developer.

## How to Update the Menu

Open assets/js/menu.js in any text editor.
Find the MENU_TABS array at the top of the file.
Each tab has an array of items.
To change a price find the item name and change the price value.
To add a new item copy an existing item in the same format and change the values.
Save the file and refresh the browser.

## How to Update Opening Hours

Search for "11:00 AM" across all files.
Update every instance you find.
The hours appear in: index.html hero eyebrow, about section badge, find us section, footer, and the JSON-LD schema in the head.

## How to Change the WhatsApp Number

Search for "918983229224" across all files.
Replace every instance with the new number.
The number must include the country code 91 without the plus sign.
